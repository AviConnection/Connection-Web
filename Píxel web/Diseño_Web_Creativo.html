PREGUNTAS FRECUENTES

Resolvemos tus dudas

¿Cuánto tiempo toma desarrollar un sitio web? +
El tiempo varía según la complejidad. Una landing page puede estar lista en 2-3 semanas, mientras que un sitio completo o e-commerce puede tomar 4-8 semanas.
¿Recibiré el código fuente completo? +
Sí, entregamos todos los archivos y credenciales necesarios. Si trabajamos con Webflow, recibirás acceso completo al proyecto.
¿Ofrecen soporte post-lanzamiento? +
Sí, todos los paquetes incluyen soporte post-lanzamiento. El tiempo varía según el plan seleccionado, desde 1 mes hasta soporte prioritario extendido.
¿Puedo solicitar modificaciones después del lanzamiento? +
Absolutamente. Ofrecemos paquetes de mantenimiento mensual para actualizaciones, ajustes y mejoras continuas de tu sitio web.
